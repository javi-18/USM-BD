La base de datos y la pagina web fue programada usando PHP y MySQL,a traves del programa Xampp

Instrucciones:
- Para la ejecución correcta del programa se debe tener carpeta que contiene los archivos php dentro de la carpeta htdocs de Xampp, el cual debe estar
instalado anteriormente, con los modulos de Apache y MySQL activados
Con esto se ingresa al administrador phpmyadmin a traves del servidor "localhost" dentro de algun navegador, el cual se le debe importar la base de datos


- Una vez instalado y cargado todo, se ejecuta la pagina dentro de algun navegador, que para efectos de prueba se ejecutó en Google Chrome, el cual no presento mayores problemas 

Supuestos:

-Las calificaciones se encuentran junto a las reseñas, ya que esto es necesario para la ejecución de ciertas consultas a la base de datos al momento de que el usurio califique alguna receta del menu
